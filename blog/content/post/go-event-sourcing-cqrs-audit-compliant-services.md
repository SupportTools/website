Event Sourcing and CQRS are architectural patterns that address a fundamental challenge in enterprise systems: maintaining a complete, immutable audit trail while delivering high read performance. Rather than persisting current state, Event Sourcing stores every state transition as an event, enabling perfect audit compliance, temporal queries, and event replay. CQRS separates the write model (commands that generate events) from the read model (projections optimized for queries). This post builds a production-grade implementation in Go with Kafka-backed event streaming.

<!--more-->

## Core Concepts

### Event Sourcing Model

In a traditional CRUD system, updating a record overwrites previous state. In Event Sourcing, every change is an immutable event appended to an event stream. The current state is derived by replaying events:

```
Command → Aggregate.Handle() → Events → Event Store
Event Store → Aggregate.Apply() → Current State
Event Store → Projector → Read Model
```

### CQRS Separation

The Command side (writes) and Query side (reads) use separate models:

- **Commands** express intent: `CreateOrder`, `CancelOrder`, `AddItem`
- **Events** record what happened: `OrderCreated`, `OrderCancelled`, `ItemAdded`
- **Aggregates** enforce business rules and generate events
- **Projections** build read-optimized views from events

### Aggregate Interface

```go
// pkg/eventsourcing/aggregate.go
package eventsourcing

// Aggregate defines the interface for all domain aggregates
type Aggregate interface {
    // GetID returns the aggregate's unique identifier
    GetID() AggregateID
    // GetType returns the aggregate type name
    GetType() string
    // GetVersion returns the current aggregate version (event count)
    GetVersion() int64
    // GetUncommittedEvents returns events generated by commands
    GetUncommittedEvents() []Event
    // ClearUncommittedEvents clears the uncommitted event buffer
    ClearUncommittedEvents()
    // Apply reconstitutes state from a historical event
    Apply(event Event) error
}

// BaseAggregate provides common aggregate functionality
type BaseAggregate struct {
    id                AggregateID
    aggregateType     string
    version           int64
    uncommitted       []Event
}

func NewBaseAggregate(id AggregateID, aggregateType string) BaseAggregate {
    return BaseAggregate{
        id:            id,
        aggregateType: aggregateType,
    }
}

func (a *BaseAggregate) GetID() AggregateID    { return a.id }
func (a *BaseAggregate) GetType() string        { return a.aggregateType }
func (a *BaseAggregate) GetVersion() int64      { return a.version }

func (a *BaseAggregate) GetUncommittedEvents() []Event {
    return a.uncommitted
}

func (a *BaseAggregate) ClearUncommittedEvents() {
    a.uncommitted = nil
}

func (a *BaseAggregate) RaiseEvent(event Event) {
    event.SequenceNum = a.version + int64(len(a.uncommitted)) + 1
    a.uncommitted = append(a.uncommitted, event)
}

func (a *BaseAggregate) IncrementVersion() {
    a.version++
}
```

## Order Domain Implementation

### Order Aggregate

```go
// internal/order/aggregate.go
package order

import (
    "errors"
    "time"

    "github.com/example/platform/pkg/eventsourcing"
    "github.com/google/uuid"
)

const OrderAggregateType = "Order"

// Order aggregate state
type Order struct {
    eventsourcing.BaseAggregate
    CustomerID string
    Status     OrderStatus
    Items      []OrderItem
    TotalCents int64
    CreatedAt  time.Time
    CancelledAt *time.Time
}

type OrderStatus string

const (
    OrderStatusPending   OrderStatus = "pending"
    OrderStatusConfirmed OrderStatus = "confirmed"
    OrderStatusShipped   OrderStatus = "shipped"
    OrderStatusCancelled OrderStatus = "cancelled"
)

type OrderItem struct {
    ProductID  string
    Name       string
    Quantity   int
    PriceCents int64
}

// Commands
type CreateOrderCommand struct {
    eventsourcing.Command
    CustomerID string
    Items      []OrderItem
}

type CancelOrderCommand struct {
    eventsourcing.Command
    Reason string
}

type ConfirmOrderCommand struct {
    eventsourcing.Command
    PaymentReference string
}

// Events
const (
    EventOrderCreated   eventsourcing.EventType = "order.created"
    EventOrderCancelled eventsourcing.EventType = "order.cancelled"
    EventOrderConfirmed eventsourcing.EventType = "order.confirmed"
)

type OrderCreatedData struct {
    CustomerID string      `json:"customer_id"`
    Items      []OrderItem `json:"items"`
    TotalCents int64       `json:"total_cents"`
}

type OrderCancelledData struct {
    Reason string `json:"reason"`
}

type OrderConfirmedData struct {
    PaymentReference string `json:"payment_reference"`
}

// NewOrder creates an empty aggregate ready for reconstitution
func NewOrder(id eventsourcing.AggregateID) *Order {
    return &Order{
        BaseAggregate: eventsourcing.NewBaseAggregate(id, OrderAggregateType),
    }
}

// Handle processes a CreateOrderCommand
func (o *Order) HandleCreate(cmd CreateOrderCommand) error {
    if len(cmd.Items) == 0 {
        return errors.New("order must have at least one item")
    }

    total := int64(0)
    for _, item := range cmd.Items {
        if item.Quantity <= 0 {
            return errors.New("item quantity must be positive")
        }
        total += int64(item.Quantity) * item.PriceCents
    }

    o.RaiseEvent(eventsourcing.Event{
        ID:            eventsourcing.EventID(uuid.NewString()),
        AggregateID:   cmd.AggregateID,
        AggregateType: OrderAggregateType,
        Type:          EventOrderCreated,
        Version:       1,
        OccurredAt:    time.Now().UTC(),
        Metadata: eventsourcing.EventMetadata{
            CorrelationID: cmd.CorrelationID,
            UserID:        cmd.UserID,
        },
        Data: OrderCreatedData{
            CustomerID: cmd.CustomerID,
            Items:      cmd.Items,
            TotalCents: total,
        },
    })
    return nil
}

// Handle processes a CancelOrderCommand
func (o *Order) HandleCancel(cmd CancelOrderCommand) error {
    if o.Status == OrderStatusCancelled {
        return errors.New("order already cancelled")
    }
    if o.Status == OrderStatusShipped {
        return errors.New("cannot cancel a shipped order")
    }

    o.RaiseEvent(eventsourcing.Event{
        ID:            eventsourcing.EventID(uuid.NewString()),
        AggregateID:   cmd.AggregateID,
        AggregateType: OrderAggregateType,
        Type:          EventOrderCancelled,
        Version:       1,
        OccurredAt:    time.Now().UTC(),
        Metadata: eventsourcing.EventMetadata{
            CorrelationID: cmd.CorrelationID,
            UserID:        cmd.UserID,
        },
        Data: OrderCancelledData{Reason: cmd.Reason},
    })
    return nil
}

// Apply reconstitutes aggregate state from a historical event
func (o *Order) Apply(event eventsourcing.Event) error {
    switch event.Type {
    case EventOrderCreated:
        data, ok := event.Data.(OrderCreatedData)
        if !ok {
            return errors.New("invalid event data for order.created")
        }
        o.CustomerID = data.CustomerID
        o.Items = data.Items
        o.TotalCents = data.TotalCents
        o.Status = OrderStatusPending
        o.CreatedAt = event.OccurredAt

    case EventOrderCancelled:
        now := event.OccurredAt
        o.Status = OrderStatusCancelled
        o.CancelledAt = &now

    case EventOrderConfirmed:
        o.Status = OrderStatusConfirmed
    }

    o.IncrementVersion()
    return nil
}
```

## Event Store Design

### Event Store Interface

```go
// pkg/eventsourcing/store.go
package eventsourcing

import (
    "context"
    "errors"
)

var (
    ErrAggregateNotFound    = errors.New("aggregate not found")
    ErrOptimisticConcurrency = errors.New("optimistic concurrency conflict")
)

// EventStore persists and retrieves event streams
type EventStore interface {
    // AppendEvents appends events to the stream for an aggregate.
    // expectedVersion is the version the caller expects the aggregate to be at.
    // Returns ErrOptimisticConcurrency if the aggregate has been modified.
    AppendEvents(ctx context.Context, aggregateID AggregateID, aggregateType string, events []Event, expectedVersion int64) error

    // LoadEvents retrieves all events for an aggregate, optionally from a start sequence.
    LoadEvents(ctx context.Context, aggregateID AggregateID, fromSequence int64) ([]Event, error)

    // LoadEventsByType retrieves events filtered by type, useful for projections.
    LoadEventsByType(ctx context.Context, eventTypes []EventType, fromGlobalSequence int64, limit int) ([]Event, error)
}
```

### PostgreSQL Event Store Implementation

```go
// pkg/eventsourcing/postgres_store.go
package eventsourcing

import (
    "context"
    "database/sql"
    "encoding/json"
    "fmt"
    "time"

    _ "github.com/lib/pq"
)

type PostgresEventStore struct {
    db *sql.DB
}

func NewPostgresEventStore(db *sql.DB) *PostgresEventStore {
    return &PostgresEventStore{db: db}
}

func (s *PostgresEventStore) AppendEvents(
    ctx context.Context,
    aggregateID AggregateID,
    aggregateType string,
    events []Event,
    expectedVersion int64,
) error {
    tx, err := s.db.BeginTx(ctx, &sql.TxOptions{Isolation: sql.LevelSerializable})
    if err != nil {
        return fmt.Errorf("begin transaction: %w", err)
    }
    defer tx.Rollback()

    // Optimistic concurrency check
    var currentVersion int64
    err = tx.QueryRowContext(ctx,
        `SELECT COALESCE(MAX(sequence_num), 0) FROM events WHERE aggregate_id = $1`,
        string(aggregateID),
    ).Scan(&currentVersion)
    if err != nil {
        return fmt.Errorf("check current version: %w", err)
    }

    if currentVersion != expectedVersion {
        return fmt.Errorf("%w: expected %d, got %d",
            ErrOptimisticConcurrency, expectedVersion, currentVersion)
    }

    // Insert events
    for _, event := range events {
        dataJSON, err := json.Marshal(event.Data)
        if err != nil {
            return fmt.Errorf("marshal event data: %w", err)
        }
        metaJSON, err := json.Marshal(event.Metadata)
        if err != nil {
            return fmt.Errorf("marshal event metadata: %w", err)
        }

        _, err = tx.ExecContext(ctx, `
            INSERT INTO events (
                id, aggregate_id, aggregate_type, event_type,
                event_version, sequence_num, occurred_at,
                metadata, data
            ) VALUES ($1, $2, $3, $4, $5, $6, $7, $8, $9)`,
            string(event.ID),
            string(aggregateID),
            aggregateType,
            string(event.Type),
            int(event.Version),
            event.SequenceNum,
            event.OccurredAt,
            metaJSON,
            dataJSON,
        )
        if err != nil {
            return fmt.Errorf("insert event: %w", err)
        }
    }

    return tx.Commit()
}

func (s *PostgresEventStore) LoadEvents(
    ctx context.Context,
    aggregateID AggregateID,
    fromSequence int64,
) ([]Event, error) {
    rows, err := s.db.QueryContext(ctx, `
        SELECT id, aggregate_id, aggregate_type, event_type,
               event_version, sequence_num, occurred_at, metadata, data
        FROM events
        WHERE aggregate_id = $1 AND sequence_num > $2
        ORDER BY sequence_num ASC`,
        string(aggregateID), fromSequence,
    )
    if err != nil {
        return nil, fmt.Errorf("query events: %w", err)
    }
    defer rows.Close()

    return s.scanEvents(rows)
}

func (s *PostgresEventStore) scanEvents(rows *sql.Rows) ([]Event, error) {
    var events []Event
    for rows.Next() {
        var e Event
        var metaJSON, dataJSON []byte
        var occurredAt time.Time

        err := rows.Scan(
            &e.ID, &e.AggregateID, &e.AggregateType,
            &e.Type, &e.Version, &e.SequenceNum,
            &occurredAt, &metaJSON, &dataJSON,
        )
        if err != nil {
            return nil, fmt.Errorf("scan event: %w", err)
        }

        e.OccurredAt = occurredAt
        if err := json.Unmarshal(metaJSON, &e.Metadata); err != nil {
            return nil, fmt.Errorf("unmarshal metadata: %w", err)
        }
        // Data is left as json.RawMessage for caller to unmarshal into concrete type
        e.Data = json.RawMessage(dataJSON)
        events = append(events, e)
    }
    return events, rows.Err()
}
```

```sql
CREATE TABLE events (
    global_seq      BIGSERIAL PRIMARY KEY,
    id              UUID NOT NULL UNIQUE,
    aggregate_id    VARCHAR(255) NOT NULL,
    aggregate_type  VARCHAR(255) NOT NULL,
    event_type      VARCHAR(255) NOT NULL,
    event_version   INT NOT NULL DEFAULT 1,
    sequence_num    BIGINT NOT NULL,
    occurred_at     TIMESTAMPTZ NOT NULL,
    metadata        JSONB NOT NULL DEFAULT '{}',
    data            JSONB NOT NULL DEFAULT '{}'
);

-- Unique constraint for optimistic concurrency
CREATE UNIQUE INDEX idx_events_aggregate_sequence
    ON events (aggregate_id, sequence_num);

-- Index for event type queries (projector reads)
CREATE INDEX idx_events_type_global_seq
    ON events (event_type, global_seq);

-- Index for aggregate loading
CREATE INDEX idx_events_aggregate_id
    ON events (aggregate_id, sequence_num ASC);

-- Snapshot table for performance optimization
CREATE TABLE snapshots (
    aggregate_id    VARCHAR(255) PRIMARY KEY,
    aggregate_type  VARCHAR(255) NOT NULL,
    version         BIGINT NOT NULL,
    snapshotted_at  TIMESTAMPTZ NOT NULL,
```

## Repository Pattern

### Aggregate Repository

```go
// pkg/eventsourcing/repository.go
package eventsourcing

import (
    "context"
    "encoding/json"
    "fmt"
)

// AggregateFactory creates a new empty aggregate instance
type AggregateFactory func(id AggregateID) Aggregate

// EventRegistry maps event types to their concrete data types for deserialization
type EventRegistry map[EventType]func() interface{}

// Repository manages aggregate persistence via the event store
type Repository struct {
    store    EventStore
    factory  AggregateFactory
    registry EventRegistry
    snapshots SnapshotStore
}

func NewRepository(
    store EventStore,
    factory AggregateFactory,
    registry EventRegistry,
    snapshots SnapshotStore,
) *Repository {
    return &Repository{
        store:    store,
        factory:  factory,
        registry: registry,
        snapshots: snapshots,
    }
}

// Load retrieves an aggregate by ID, reconstituting it from its event stream
func (r *Repository) Load(ctx context.Context, id AggregateID) (Aggregate, error) {
    aggregate := r.factory(id)

    // Try to load from snapshot first
    startSequence := int64(0)
    if r.snapshots != nil {
        snap, err := r.snapshots.Load(ctx, id)
        if err == nil && snap != nil {
            if err := r.applySnapshot(aggregate, snap); err != nil {
                return nil, fmt.Errorf("apply snapshot: %w", err)
            }
            startSequence = aggregate.GetVersion()
        }
    }

    // Load events after snapshot
    events, err := r.store.LoadEvents(ctx, id, startSequence)
    if err != nil {
        return nil, fmt.Errorf("load events: %w", err)
    }

    if len(events) == 0 && startSequence == 0 {
        return nil, ErrAggregateNotFound
    }

    // Apply each event to reconstitute state
    for _, event := range events {
        // Deserialize event data to concrete type
        if factory, ok := r.registry[event.Type]; ok {
            concreteData := factory()
            rawData, ok := event.Data.(json.RawMessage)
            if ok {
                if err := json.Unmarshal(rawData, concreteData); err != nil {
                    return nil, fmt.Errorf("deserialize event %s: %w", event.Type, err)
                }
                event.Data = concreteData
            }
        }

        if err := aggregate.Apply(event); err != nil {
            return nil, fmt.Errorf("apply event %s seq %d: %w",
                event.Type, event.SequenceNum, err)
        }
    }

    return aggregate, nil
}

// Save persists uncommitted events from an aggregate
func (r *Repository) Save(ctx context.Context, aggregate Aggregate) error {
    events := aggregate.GetUncommittedEvents()
    if len(events) == 0 {
        return nil
    }

    // Expected version is current version minus uncommitted events
    expectedVersion := aggregate.GetVersion()

    if err := r.store.AppendEvents(ctx,
        aggregate.GetID(),
        aggregate.GetType(),
        events,
        expectedVersion,
    ); err != nil {
        return fmt.Errorf("append events: %w", err)
    }

    aggregate.ClearUncommittedEvents()
    return nil
}
```

## Snapshot Support

Snapshotting prevents expensive event replay for long-lived aggregates:

```go
// pkg/eventsourcing/snapshot.go
package eventsourcing

import (
    "context"
    "encoding/json"
    "time"
)

type Snapshot struct {
    AggregateID   AggregateID
    AggregateType string
    Version       int64
    SnapshottedAt time.Time
    Data          json.RawMessage
}

type SnapshotStore interface {
    Save(ctx context.Context, snap Snapshot) error
    Load(ctx context.Context, id AggregateID) (*Snapshot, error)
}

// Snapshottable aggregates can serialize and deserialize their state
type Snapshottable interface {
    TakeSnapshot() (json.RawMessage, error)
    RestoreSnapshot(data json.RawMessage) error
}

// SnapshotPolicy determines when to take a snapshot
type SnapshotPolicy interface {
    ShouldSnapshot(aggregate Aggregate) bool
}

// EveryNEventsPolicy snapshots every N events
type EveryNEventsPolicy struct {
    N int64
}

func (p *EveryNEventsPolicy) ShouldSnapshot(aggregate Aggregate) bool {
    return aggregate.GetVersion()%p.N == 0
}
```

## Projections and Read Models

### Projection Interface

```go
// pkg/eventsourcing/projection.go
package eventsourcing

import "context"

// Projection builds a read model from events
type Projection interface {
    // EventTypes returns the event types this projection handles
    EventTypes() []EventType
    // HandleEvent processes an event and updates the read model
    HandleEvent(ctx context.Context, event Event) error
}

// ProjectionRunner manages projection state and event consumption
type ProjectionRunner struct {
    projection  Projection
    store       EventStore
    checkpoints CheckpointStore
    name        string
}

func (r *ProjectionRunner) Run(ctx context.Context) error {
    // Load last processed global sequence
    lastSeq, err := r.checkpoints.Load(ctx, r.name)
    if err != nil {
        lastSeq = 0
    }

    for {
        events, err := r.store.LoadEventsByType(ctx,
            r.projection.EventTypes(),
            lastSeq,
            100, // batch size
        )
        if err != nil {
            return err
        }

        for _, event := range events {
            if err := r.projection.HandleEvent(ctx, event); err != nil {
                return err
            }
            lastSeq = event.SequenceNum
        }

        if len(events) < 100 {
            // Save checkpoint and wait for new events
            if err := r.checkpoints.Save(ctx, r.name, lastSeq); err != nil {
                return err
            }
            // In production, use a subscription/notification mechanism
            return nil
        }
    }
}
```

### Order Summary Projection

```go
// internal/order/projections/order_summary.go
package projections

import (
    "context"
    "database/sql"
    "encoding/json"
    "fmt"

    "github.com/example/platform/internal/order"
    "github.com/example/platform/pkg/eventsourcing"
)

// OrderSummaryProjection builds a queryable orders table
type OrderSummaryProjection struct {
    db *sql.DB
}

func (p *OrderSummaryProjection) EventTypes() []eventsourcing.EventType {
    return []eventsourcing.EventType{
        order.EventOrderCreated,
        order.EventOrderCancelled,
        order.EventOrderConfirmed,
    }
}

func (p *OrderSummaryProjection) HandleEvent(ctx context.Context, event eventsourcing.Event) error {
    switch event.Type {
    case order.EventOrderCreated:
        return p.handleOrderCreated(ctx, event)
    case order.EventOrderCancelled:
        return p.handleOrderCancelled(ctx, event)
    case order.EventOrderConfirmed:
        return p.handleOrderConfirmed(ctx, event)
    }
    return nil
}

func (p *OrderSummaryProjection) handleOrderCreated(ctx context.Context, event eventsourcing.Event) error {
    data, err := unmarshalData[order.OrderCreatedData](event)
    if err != nil {
        return err
    }

    _, err = p.db.ExecContext(ctx, `
        INSERT INTO order_summaries (
            id, customer_id, status, total_cents, item_count, created_at
        ) VALUES ($1, $2, $3, $4, $5, $6)
        ON CONFLICT (id) DO NOTHING`,
        string(event.AggregateID),
        data.CustomerID,
        "pending",
        data.TotalCents,
        len(data.Items),
        event.OccurredAt,
    )
    return err
}

func (p *OrderSummaryProjection) handleOrderCancelled(ctx context.Context, event eventsourcing.Event) error {
    _, err := p.db.ExecContext(ctx, `
        UPDATE order_summaries
        SET status = 'cancelled', updated_at = $2
        WHERE id = $1`,
        string(event.AggregateID),
        event.OccurredAt,
    )
    return err
}

func (p *OrderSummaryProjection) handleOrderConfirmed(ctx context.Context, event eventsourcing.Event) error {
    _, err := p.db.ExecContext(ctx, `
        UPDATE order_summaries
        SET status = 'confirmed', updated_at = $2
        WHERE id = $1`,
        string(event.AggregateID),
        event.OccurredAt,
    )
    return err
}

func unmarshalData[T any](event eventsourcing.Event) (T, error) {
    var result T
    raw, ok := event.Data.(json.RawMessage)
    if !ok {
        return result, fmt.Errorf("expected json.RawMessage, got %T", event.Data)
    }
    return result, json.Unmarshal(raw, &result)
}
```

## Kafka Integration for Event Streaming

Publishing events to Kafka enables cross-service event consumption:

```go
// pkg/eventsourcing/kafka_publisher.go
package eventsourcing

import (
    "context"
    "encoding/json"
    "fmt"

    "github.com/twmb/franz-go/pkg/kgo"
)

type KafkaEventPublisher struct {
    client *kgo.Client
    topic  string
}

func NewKafkaEventPublisher(brokers []string, topic string) (*KafkaEventPublisher, error) {
    client, err := kgo.NewClient(
        kgo.SeedBrokers(brokers...),
        kgo.RequiredAcks(kgo.AllISRAcks()),
        kgo.ProducerBatchMaxBytes(1_000_000),
        kgo.RecordRetries(5),
    )
    if err != nil {
        return nil, fmt.Errorf("create kafka client: %w", err)
    }
    return &KafkaEventPublisher{client: client, topic: topic}, nil
}

func (p *KafkaEventPublisher) Publish(ctx context.Context, events []Event) error {
    records := make([]*kgo.Record, len(events))
    for i, event := range events {
        data, err := json.Marshal(event)
        if err != nil {
            return fmt.Errorf("marshal event: %w", err)
        }
        records[i] = &kgo.Record{
            Topic:   p.topic,
            Key:     []byte(event.AggregateID),
            Value:   data,
            Headers: []kgo.RecordHeader{
                {Key: "event-type", Value: []byte(event.Type)},
                {Key: "aggregate-type", Value: []byte(event.AggregateType)},
            },
        }
    }

    results := p.client.ProduceSync(ctx, records...)
    for _, result := range results {
        if result.Err != nil {
            return fmt.Errorf("produce event: %w", result.Err)
        }
    }
    return nil
}
```

### Transactional Outbox Pattern

For guaranteed event delivery without distributed transactions, use the outbox pattern:

```go
// pkg/eventsourcing/outbox.go
package eventsourcing

import (
    "context"
    "database/sql"
    "encoding/json"
    "time"
)

// OutboxEvent is stored in the same transaction as the aggregate events
type OutboxEvent struct {
    ID          string
    AggregateID string
    EventType   string
    Payload     json.RawMessage
    CreatedAt   time.Time
    ProcessedAt *time.Time
}

// TransactionalOutbox writes events and outbox records atomically
type TransactionalOutbox struct {
    db        *sql.DB
    eventStore EventStore
    publisher  EventPublisher
}

func (o *TransactionalOutbox) AppendAndPublish(
    ctx context.Context,
    tx *sql.Tx,
    aggregateID AggregateID,
    aggregateType string,
    events []Event,
    expectedVersion int64,
) error {
    // 1. Append to event store (within the same transaction)
    if err := o.eventStore.AppendEvents(ctx, aggregateID, aggregateType, events, expectedVersion); err != nil {
        return err
    }

    // 2. Write to outbox table atomically
    for _, event := range events {
        payload, err := json.Marshal(event)
        if err != nil {
            return err
        }
        _, err = tx.ExecContext(ctx, `
            INSERT INTO outbox_events (id, aggregate_id, event_type, payload, created_at)
            VALUES ($1, $2, $3, $4, NOW())`,
            string(event.ID),
            string(aggregateID),
            string(event.Type),
            payload,
        )
        if err != nil {
            return err
        }
    }
    return nil
}
```

## NATS JetStream Integration

For lower-latency event streaming, NATS JetStream provides at-least-once delivery:

```go
// pkg/eventsourcing/nats_publisher.go
package eventsourcing

import (
    "context"
    "encoding/json"
    "fmt"

    "github.com/nats-io/nats.go"
    "github.com/nats-io/nats.go/jetstream"
)

type NATSEventPublisher struct {
    js      jetstream.JetStream
    subject string
}

func NewNATSEventPublisher(nc *nats.Conn, streamName, subject string) (*NATSEventPublisher, error) {
    js, err := jetstream.New(nc)
    if err != nil {
        return nil, fmt.Errorf("create JetStream context: %w", err)
    }

    ctx := context.Background()
    _, err = js.CreateOrUpdateStream(ctx, jetstream.StreamConfig{
        Name:     streamName,
        Subjects: []string{subject + ".>"},
        Storage:  jetstream.FileStorage,
        Replicas: 3,
        Retention: jetstream.WorkQueuePolicy,
    })
    if err != nil {
        return nil, fmt.Errorf("create stream: %w", err)
    }

    return &NATSEventPublisher{js: js, subject: subject}, nil
}

func (p *NATSEventPublisher) Publish(ctx context.Context, events []Event) error {
    for _, event := range events {
        data, err := json.Marshal(event)
        if err != nil {
            return fmt.Errorf("marshal event: %w", err)
        }

        subject := fmt.Sprintf("%s.%s.%s",
            p.subject,
            event.AggregateType,
            string(event.Type),
        )

        _, err = p.js.Publish(ctx, subject, data,
            jetstream.WithMsgID(string(event.ID)),
        )
        if err != nil {
            return fmt.Errorf("publish event: %w", err)
        }
    }
    return nil
}
```

## Handling Eventual Consistency

CQRS read models are eventually consistent with the write model. Production systems must handle this gracefully.

### Version Token Pattern

Return a version token with write responses for clients to poll:

```go
// internal/order/handler.go
package order

import (
    "context"
    "net/http"
    "time"
)

type CreateOrderResponse struct {
    OrderID      string `json:"order_id"`
    VersionToken string `json:"version_token"`
}

type VersionToken struct {
    AggregateID string    `json:"aggregate_id"`
    Version     int64     `json:"version"`
    WrittenAt   time.Time `json:"written_at"`
}

// After writing, client polls GET /orders/{id}?min_version={token}
// until the read model reflects the write
func (h *Handler) WaitForProjection(
    ctx context.Context,
    aggregateID string,
    minVersion int64,
    timeout time.Duration,
) error {
    deadline := time.Now().Add(timeout)
    for time.Now().Before(deadline) {
        var currentVersion int64
        err := h.db.QueryRowContext(ctx,
            "SELECT version FROM order_summaries WHERE id = $1",
            aggregateID,
        ).Scan(&currentVersion)

        if err == nil && currentVersion >= minVersion {
            return nil
        }

        select {
        case <-ctx.Done():
            return ctx.Err()
        case <-time.After(50 * time.Millisecond):
        }
    }
    return fmt.Errorf("projection not caught up after %v", timeout)
}
```

## Temporal Queries and Audit Trail

One of the key benefits of Event Sourcing is the complete audit trail:

```go
// pkg/eventsourcing/audit.go
package eventsourcing

import (
    "context"
    "time"
)

// TemporalQuery retrieves aggregate state at a specific point in time
func TemporalQuery(
    ctx context.Context,
    store EventStore,
    factory AggregateFactory,
    registry EventRegistry,
    id AggregateID,
    asOf time.Time,
) (Aggregate, error) {
    events, err := store.LoadEvents(ctx, id, 0)
    if err != nil {
        return nil, err
    }

    aggregate := factory(id)
    for _, event := range events {
        if event.OccurredAt.After(asOf) {
            break // Stop at the temporal boundary
        }

        // Deserialize and apply
        if factory, ok := registry[event.Type]; ok {
            concreteData := factory()
            if raw, ok := event.Data.(json.RawMessage); ok {
                json.Unmarshal(raw, concreteData)
                event.Data = concreteData
            }
        }
        aggregate.Apply(event)
    }
    return aggregate, nil
}
```

## Summary

Event Sourcing and CQRS in Go provide a robust foundation for audit-compliant services. The patterns covered — aggregate design with optimistic concurrency, PostgreSQL-backed event stores, snapshot optimization, read-model projections, and Kafka/NATS event streaming — compose into a complete production system. The transactional outbox pattern ensures reliable event delivery without distributed transactions, while the version token pattern handles eventual consistency transparently in API contracts. Together these patterns deliver complete auditability, temporal query capability, and the scalability benefits of separated read and write paths.
